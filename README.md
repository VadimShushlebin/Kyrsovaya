# Курсовой проект: База данных "RPA системы"
Файловая база данных – это файл, в котором хранятся упорядоченные записи данных, описывающих заданную предметную область. Для работы с ними необходимо реализовать программу, позволяющую создавать новые записи, выполнять поиск по заданным критериям.

Целью данной работы является разработка программы для реализации файловой базы данных с заданным набором функций.

Для достижения поставленной цели, необходимо решить следующие задачи:

1.	Реализовать структуру данных и определить подходящие типы данных для каждого поля.

2.	Реализовать простой и понятный интерфейс для взаимодействия пользователя с программой, который будет работать пока пользователь не захочет выйти из программы.

3.	Реализовать сохранение данных в бинарный файл и загрузку данных из бинарного файла

4.	Реализовать поиск записей в заданном интервале значений

5.	Реализовать сортировку с предварительным математическим преобразованием значений

## Выполняемые программой операции

1. Загрузка данных из файла – загрузка базы данных RPA-систем из файла rpa_systems.txt.
2. Просмотр всех записей – вывод всей базы данных RPA-систем на экран.
3. Поиск по заданному диапазону – поиск систем по диапазону стоимости лицензии.
4. Сортировка данных – сортировка всей базы данных по кубическому корню точности выполнения.
5. Добавление новой записи – ввод информации о новой RPA-системе.
6. Сохранение данных в файл – сохранение имеющихся данных в файл rpa_systems.txt.
7. Заполнение случайными данными – добавление записей со случайными значениями.
8. Выход – завершение работы программы

## Описание функций программмы 

1. int show_menu()

Назначение: Отображает текстовое меню с доступными действиями и получает выбор пользователя.

Возвращает: Целое число (int), соответствующее выбранному пункту меню (от 1 до 8).

2. rpa_sys input_data()

Назначение: Интерактивно запрашивает у пользователя данные для заполнения одной записи.

Возвращает: Заполненную структуру rpa_sys с данными, введенными пользователем.

3. void output_data(rpa_sys system)

Назначение: Выводит на экран все поля записи.

Параметры:

* system - Структура rpa_sys, данные которой нужно отобразить.

4. int load_from_file(rpa_sys* systems)

Назначение: Загружает массив записей из бинарного файла "rpa_systems.txt" в память.

Параметры:

* systems: Указатель на массив структур rpa_sys.

Возвращает: количество успешно загруженных записей.

5. void view_all_records(rpa_sys* systems, int count)

Назначение: Отображает все записи, находящиеся в оперативной памяти.

Параметры:

* systems - Указатель на массив структур rpa_sys.

* count - Количество актуальных записей в массиве.

6. void search_by_range(rpa_sys* systems, int count)

Назначение: Осуществляет поиск и вывод записей, у которых значение поля cost (стоимость) попадает в указанный пользователем диапазон.

Параметры:

* systems: Указатель на массив структур rpa_sys.

* count: Количество записей в массиве.

Возвращает: количество найденных записей found.

7. int compare(const void* a, const void* b)

Назначение: Функция сравнения для qsort. Сравнивает две записи по кубическому корню от их точности. Используется для сортировки.

Параметры:

* a, b: Указатели const void* на сравниваемые элементы массива.

8. void sort_data(rpa_sys* systems, int count)

Назначение: Сортирует массив записей по убыванию кубического корня от точности, используя стандартную функцию qsort и функцию сравнения compare.

Параметры:

* systems: Указатель на массив структур rpa_sys.

* count: Количество актуальных записей в массиве.

9. int add_new_record(rpa_sys* systems, int count)

Назначение: Добавляет одну новую запись в массив. Данные для записи запрашиваются у пользователя через функцию input_data.

Параметры:

* systems: Указатель на массив структур rpa_sys.

* count: Текущее количество актуальных записей в массиве.

Возвращает: Новое количество записей в массиве после добавления.

10. int save_to_file(rpa_sys* systems, int count)

Назначение: Сохраняет все текущие записи из оперативной памяти в бинарный файл "rpa_systems.txt".

Параметры:

* systems: Указатель на массив структур rpa_sys.

* count: Количество актуальных записей в массиве, которые необходимо сохранить.

Возвращает: количество сохраненных записей.

11. int generate_random_data(rpa_sys* systems, int count)

Назначение: Генерирует указанное количество записей со случайными данными и добавляет их в массив.

Параметры:

* systems: Указатель на массив структур rpa_sys.

* count: Текущее количество актуальных записей в массиве.

Возвращает: Новое количество записей в массиве после добавления сгенерированных.
