# Курсовой проект: База данных "RPA системы"
## Выполняемые программой операции
1. Загрузка данных из файла – загрузка базы данных RPA-систем из файла rpa_systems.txt.
2. Просмотр всех записей – вывод всей базы данных RPA-систем на экран.
3. Поиск по заданному диапазону – поиск систем по диапазону стоимости лицензии.
4. Сортировка данных – сортировка всей базы данных по кубическому корню точности выполнения.
5. Добавление новой записи – ввод информации о новой RPA-системе.
6. Сохранение данных в файл – сохранение имеющихся данных в файл rpa_systems.txt.
7. Заполнение случайными данными – добавление записей со случайными значениями.
8. Выход – завершение работы программы
