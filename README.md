# Курсовой проект: База данных "RPA системы"
## Выполняемые программой операции
1. Загрузка данных из файла – загрузка базы данных RPA-систем из файла rpa_systems.txt.
2. Просмотр всех записей – вывод всей базы данных RPA-систем на экран.
3. Поиск по заданному диапазону – поиск систем по диапазону стоимости лицензии.
4. Сортировка данных – сортировка всей базы данных по кубическому корню точности выполнения.
5. Добавление новой записи – ввод информации о новой RPA-системе.
6. Сохранение данных в файл – сохранение имеющихся данных в файл rpa_systems.txt.
7. Заполнение случайными данными – добавление записей со случайными значениями.
8. Выход – завершение работы программы

## Описание функций программмы 

1. int show_menu()
Назначение: Отображает текстовое меню с доступными действиями и получает выбор пользователя.
Возвращает: Целое число (int), соответствующее выбранному пункту меню (от 1 до 8).

2. rpa_sys input_data()
Назначение: Интерактивно запрашивает у пользователя данные для заполнения одной записи.
Возвращает: Заполненную структуру rpa_sys с данными, введенными пользователем.

3. void output_data(rpa_sys system)
Назначение: Выводит на экран все поля записи.
Параметры:
system: Структура rpa_sys, данные которой нужно отобразить.

5. int load_from_file(rpa_sys* systems)
Назначение: Загружает массив записей из бинарного файла "rpa_systems.txt" в память.
Параметры:
* systems: Указатель на массив структур rpa_sys.
Возвращает: количество успешно загруженных записей.

6. void view_all_records(rpa_sys* systems, int count)
Назначение: Отображает все записи, находящиеся в оперативной памяти.
Параметры:

systems: Указатель на массив структур rpa_sys.

count: Количество актуальных записей в массиве.
Возвращает: Нет (void).

7. void search_by_range(rpa_sys* systems, int count)
Назначение: Осуществляет поиск и вывод записей, у которых значение поля cost (стоимость) попадает в указанный пользователем диапазон.
Параметры:

systems: Указатель на массив структур rpa_sys.

count: Количество актуальных записей в массиве.
Возвращает: Нет (void).

8. int compare(const void* a, const void* b)
Назначение: Функция сравнения для qsort. Сравнивает две записи по кубическому корню от их точности (accuracy). Используется для сортировки по убыванию данного показателя.
Параметры:

a, b: Указатели const void* на сравниваемые элементы массива (в данном случае на структуры rpa_sys).
Возвращает:

1, если кубический корень точности a меньше b (для сортировки по убыванию).

-1, если кубический корень точности a больше b.

0, если они равны.

9. void sort_data(rpa_sys* systems, int count)
Назначение: Сортирует массив записей по убыванию кубического корня от точности, используя стандартную функцию qsort и функцию сравнения compare.
Параметры:

systems: Указатель на массив структур rpa_sys.

count: Количество актуальных записей в массиве.
Возвращает: Нет (void).

10. int add_new_record(rpa_sys* systems, int count)
Назначение: Добавляет одну новую запись в массив. Данные для записи запрашиваются у пользователя через функцию input_data.
Параметры:

systems: Указатель на массив структур rpa_sys.

count: Текущее количество актуальных записей в массиве.
Возвращает: Новое количество записей в массиве (int) после добавления.

11. int save_to_file(rpa_sys* systems, int count)
Назначение: Сохраняет все текущие записи из оперативной памяти в бинарный файл "rpa_systems.txt".
Параметры:

systems: Указатель на массив структур rpa_sys.

count: Количество актуальных записей в массиве, которые необходимо сохранить.
Возвращает: Целое число (int): 1 в случае успешного сохранения всех записей, 0 в случае ошибки.

12. int getRandomNumber(int min, int maxxx)
Назначение: Генерирует случайное целое число в заданном диапазоне [min, maxxx].
Параметры:

min: Нижняя граница диапазона.

maxxx: Верхняя граница диапазона.
Возвращает: Случайное целое число (int) в указанном диапазоне.

13. void generate_random_name(char* name)
Назначение: Заполняет переданную строку одним из четырех заранее заданных случайных названий RPA-систем.
Параметры:

name: Указатель на массив char (строку), который будет заполнен случайным именем.
Возвращает: Нет (void).

14. void generate_random_monitor(char* monitor)
Назначение: Заполняет переданную строку одним из четырех заранее заданных случайных описаний возможностей мониторинга.
Параметры:

monitor: Указатель на массив char (строку), который будет заполнен случайным описанием.
Возвращает: Нет (void).

15. void generate_random_erp(char* erp)
Назначение: Заполняет переданную строку одним из четырех заранее заданных случайных названий ERP-систем.
Параметры:

erp: Указатель на массив char (строку), который будет заполнен случайным названием ERP.
Возвращает: Нет (void).

16. int generate_random_data(rpa_sys* systems, int count)
Назначение: Генерирует указанное количество записей со случайными данными и добавляет их в массив. Использует вспомогательные функции для генерации строковых полей.
Параметры:

systems: Указатель на массив структур rpa_sys.

count: Текущее количество актуальных записей в массиве.
Возвращает: Новое количество записей в массиве (int) после добавления сгенерированных.
